== Description ==
WPDev Bottom to Top plugin enables a Back to Top button on your WordPress website, to enhance user experience by providing easy navigation back to the top. Compatible with all major browsers and works seamlessly with all themes, including responsive designs.

Plugin Documentation: [Read more here](https://wordpress.org/plugins/wpdev-bottom-to-top/).

== ⚡ You can change settings ==
* No setup needed.
* Unlimited colors for customization.
* Easily customize button appearance.

* Just activate the plugin to enable its features.

== ⚡ Features ==
* Ready to use without any setup.

== Installation ==
= Minimum Requirements =
* WordPress 4.5 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater

= AUTOMATIC INSTALLATION =
1. Log in to your WordPress dashboard.
2. Navigate to the Plugins menu and click Add New.
3. Search for WPDev Bottom To Top and click Install Now.

= MANUAL INSTALLATION =
1. Upload `wpdev-bottom-to-top` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

= UPDATING =
Automatic updates should work seamlessly. Always backup your website before updates.
